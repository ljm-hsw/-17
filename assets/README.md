# 资源文件说明

本小程序需要以下图片资源，请将对应图片放入此目录：

## 根目录 assets/
- scu-logo.png          四川大学校徽（圆形，120x120px以上）
- campus-jiangan.jpg     江安校区风景大图（首页顶部和登录页背景，建议750x420px）
- avatar-default.png     默认用户头像
- wristband.png          智能手环实物图（绑卡页展示用）
- campus-map-comic.jpg   漫画风江安校区地图（校园导览页，建议750x750px）

## assets/spots/ 景点图片
- library.jpg            江安图书馆
- mingyuan-lake.jpg      明远湖
- long-bridge.jpg        长桥
- knowledge-square.jpg   知识广场
- sports-center.jpg      体育中心
- east-garden.jpg        东园

## 注意事项
1. 图片建议使用 jpg 格式（照片）或 png 格式（logo/图标）
2. 首页英雄图建议比例 16:9，宽度 750px
3. 地图图片建议正方形，宽度 750px
4. 景点图片建议比例 16:10 或 4:3
