## Package Preservation

- Preserve-only：`[Content_Types].xml` 中未受新增图片影响的声明、`docProps/app.xml`、主题、未修改样式定义、编号定义、页脚关系、模板原有页码字段和未编辑关系。
- Editable：`word/document.xml`、作品名所在 header、`word/settings.xml` 的字段更新标志、因替换/新增图片而变化的 media 与 relationships、必要的 core metadata。
- 删除填写说明和云计算示例时允许 `document.xml`、相关图片关系与 media 发生有解释的变化；其他包部件不得无故消失。
- 参考文件在整个制作过程中保持字节级不变，所有生成操作仅作用于工作副本和最终文件。

## Fidelity Gates

- A4 尺寸、页边距、正文版心、页眉页脚和页码系统必须保持模板来源特征。
- 封面信息表、摘要、目录和六章结构完整；填写说明和所有示例内容为零。
- 复用模板标题层级，不出现手工伪标题或四级标题。
- 最终 DOCX 与参考 DOCX 分别渲染后，允许正文内容和分页变化，但不得出现无解释的页面几何、页眉页脚或样式体系变化。
- LibreOffice 若不能解析模板的 SimSun 字体，工作副本可将东亚字体映射为宋体等价的 `Songti SC`，字号、粗细、行距和其他模板属性保持不变，以确保最终逐页渲染不缺字。
